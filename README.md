# Allegiant Air — Databricks Assets

This repository holds Databricks notebooks and supporting material shared with Allegiant Air.

## Contents

| Folder | Description |
|---|---|
| [`allegiant-json-pii-masking`](allegiant-json-pii-masking/) | Configurable, schema-preserving PII masking **inside** nested JSON on Databricks — masks only the sensitive values (a `name`, or a first/last name embedded in a URL) while preserving the full JSON structure and all non-PII data. See the folder's [README](allegiant-json-pii-masking/README.md) for full details. |

Each folder contains its own README with setup, usage, and configuration details.
